# React To Do List Assignment

React ToDo List -  
Create a ToDo list project utilizing the concepts we have learned in React thus far!

A ToDo item should consist of an id (unique), a task name, a description, and a completed status (boolean).

The application user should be able to enter their todo item into a form and upon submission, it should be added to the list.

The user should have the ability to complete tasks, delete tasks, and update tasks (we're going to go over update together but feel free to try it on your own!).

You are free to design the task however you see fit, but it should showcase the task name, description, and an indicator for whether it has been completed or not.

Feel free to utilize bootstrap/CSS to improve the design of the project, but it is not required to do so

## STRETCH GOALS:

- add the ability to delete all completed tasks

- Simulate what it's like to "login" to an application by adding a username property. You can achieve this by adding a username property to your tasks and including the ability for a user to "sign in" by entering a username into a form. The user should only see tasks associated with their username.

- Add a priority level to your tasks: LOW, MEDIUM, and HIGH that can be selected using the select and option tags.  After getting this, try adding a feature to sort by Priority.

- anything else you can think of !!!
